Home Assistant Google Authentication

This application is a personal Home Assistant integration that allows a user to connect their Google account to their own Home Assistant instance.

The purpose of this application is to enable Google-based services such as Google Calendar, Google Assistant, and media services to function inside Home Assistant.

Google authentication is required to securely access the user’s own Google data and provide smart home automation features such as presence detection, calendar synchronization, and voice control.

This application is intended for personal, non-commercial use. User data is not shared with third parties and remains within the user’s Home Assistant environment.
